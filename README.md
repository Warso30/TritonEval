# TritonEval
This repo is used to evaluate the performance of triton
## Setup
```bash
git clone https://github.com/Warso30/TritonEval.git
cd TritonEval
conda create -n myEnv python=3.12
conda activate myEnv
pip install -r requirements.txt
```
